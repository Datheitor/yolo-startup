Criação de banco de dados

Temos como foco a gestão de infraestrutura de banco de dados, atuando desde a análise dos servidores de banco de dados como aplicações, até o planejamento da segurança dos dados, base e permissão de acesso.

Algumas das tarefas :

  • Instalação;
  • Configuração;
  • Upgrade;
  • Monitoramento;
  • Políticas de Acesso;
  • Diagnóstico de Ambiente de Banco de Dados;
  • Procedimento de Backup e Manutenção Backups;
  • Documentação das Bases de Dados;
  • Reorganização de objetos;
  • Análise dos servidores de banco de dados: ocupação dos discos, uso de memória, swap e CPU;
  • Capacidade de processamento;
  • Duplicação e criação de ambientes para testes e desenvolvimento;

    • Essas são apenas algumas das atividades básicas da administração de banco de dados.
